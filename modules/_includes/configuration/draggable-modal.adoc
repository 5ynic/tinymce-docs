[#draggable_modal]
== draggable_modal

Use the `draggable_modal` option to enable dragging for link:{modulesDir}/ui-components/dialog/[modal dialogs].

*Type:* `Boolean`

*Default Value:* `false`

*Possible Values:* `true`, `false`

[discrete#example]
===== Example

`js
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  draggable_modal: true
});
`
