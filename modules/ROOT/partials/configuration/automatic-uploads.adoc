[[automatic_uploads]]
== automatic_uploads

Enable or disable TinyMCE from automatically uploading local images.

*Type:* `Boolean`

*Default Value:* `true`

*Possible Values:* `true`, `false`

[discrete]
[[example]]
===== Example

`js
tinymce.init({
  selector: 'textarea',  // change this value according to your HTML
  automatic_uploads: false
});
`
