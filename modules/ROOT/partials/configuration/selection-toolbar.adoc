[[selection_toolbar]]
== selection_toolbar

The enables you to specify toolbar items to include in the inlite themes text selection toolbar. We recommend that you only have formatting related controls in this toolbar but nothing is restricting you to use any of the available link:{rootDir}advanced/editor-control-identifiers/#toolbarcontrols[toolbar controls].

*Type:* `String`

*Defaults:* `bold italic | quicklink h2 h3 blockquote`

[[example]]
===== Example

[source,js]
----
tinymce.init({
  selector: 'div.tinymce',
  theme: 'inlite',
  inline: true,
  selection_toolbar: 'bold italic | quicklink h2 h3 blockquote'
});
----
