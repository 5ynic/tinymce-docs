= wordcount

This plugin adds word count functionality to TinyMCE.

[[initialization-example]]
== Initialization Example
anchor:initializationexample[historical anchor]

[source,js]
----
tinyMCE.init({
  theme : "advanced",
  mode : "textareas",
  plugins : "wordcount"
});
----
