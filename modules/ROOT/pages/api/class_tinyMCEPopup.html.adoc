= tinyMCEPopup

|===
|  |

| Class
| tinyMCEPopup

| Type
| Singleton
|===

TinyMCE popup/dialog helper class. This gives you easy access to the parent editor instance and a bunch of other things. It's higly recommended that you load this script into your dialogs.
