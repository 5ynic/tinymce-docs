= tinymce.html.Serializer

This class is used to serialize down the DOM tree into a string using a Writer instance.

[[examples]]
== Examples

[source,prettyprint]
----
new tinymce.html.Serializer().serialize(new tinymce.html.DomParser().parse('<p>text</p>'));
----
