= Advanced source code editing
:controls: toolbar button, menu item
:description: An IDE-like code editor for TinyMCE.
:keywords: code advcode codemirror
:title_nav: Advanced source code editing

The link:{rootDir}plugins/advcode.html[Advanced Code Editor] plugin (`advcode`) brings a more advanced code editor to {productname}. This code editor makes it easier to modify the HTML, and it's a very useful add-on for power users. It comes with many features often found in IDEs, all enabled by default:

* Syntax color highlighting
* Bracket matching
* Code folding
* Multiple selections/carets
